# Student Dashboard Pro

## Overview

Student Dashboard Pro is a Placement Management System built using Python, Streamlit, SQLite and Plotly.

## Features

- Admin Login
- Student Management
- Company Management
- Placement Drives
- Dashboard Analytics
- Reports

## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- Plotly

## Installation

```bash
pip install -r requirements.txt
python database.py
streamlit run app.py
```
